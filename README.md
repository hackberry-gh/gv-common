# GV::Common

Green Valley Shared Library

## Installation

Add this line to your application's Gemfile:

    gem 'gv-common'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install gv-common

## Usage

include 'gv/common/$libname'
